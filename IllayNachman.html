BISTrong

"Bistrong" is a comprehensive web application designed for tracking and training soldiers in the Israeli Air Force pilot course.

See the Code

Project Overview

The app provides detailed information about various workout drills and practices, allowing soldiers to stay on top of their physical and mental training. With Bistrong, soldiers can stay in touch with their coaches, get instant feedback, and track their progress over time.

To run the code you will need to set up a Flask Server connected to a SQL server*.
Full setup steps explained in the readme.md file
*The code currently works only with mysql servers
