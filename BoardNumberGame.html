Project Overview

An Android game in Java using Android Studio that presents players with a scrambled board of tiles, each with a number on it. The objective of the game is to arrange the numbers in order, which is a simple yet challenging task that requires players to think strategically and use their problem-solving skills. The game provides an engaging and interactive experience, making it a great way to pass the time while exercising the brain.

To play the game, users simply need to choose which board theme they want, then swipe the tiles to move them around and align the numbers in order. The game offers different levels of difficulty to accommodate players of varying skill levels. Additionally, the game features a scoring system that allows players to compete against each other and track their progress as they continue to play. Overall, the game is a fun and addictive way to challenge the mind and pass the time, and it provides an entertaining and engaging experience that will keep players coming back for more.
